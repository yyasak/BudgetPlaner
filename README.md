



### **Haushaltsplaner**



#### **Projektbeschreibung**



Der Haushaltsplaner ist eine Webapp zur Verwaltung persönlicher Einnahmen und Ausgaben.  
Benutzer können ihre finanziellen Einträge erfassen, kategorisieren und auswerten.

Ziel der Anwendung ist es, einen besseren Überblick über die eigenen Finanzen zu erhalten und Sparpotenziale zu erkennen.





#### **Funktionen**



**Budgetplanung**

* &#x20;Erfassung des monatlichen Einkommen
* &#x20;Erfassung einer Sparreserve
* &#x20;Erfassung von Fixkosten
* &#x20;Dynamisches Hinzufügen eigener Fixkosten
* &#x20;Erfassung variabler Kosten
* &#x20;Dynamisches Hinzufügen eigener variabler Kosten



**Analysefunktionen**

* &#x20;Berechnung des verfügbaren Restgeldes
* &#x20;Budgetverbrauchsanalyse
* &#x20;Sparquote
* &#x20;Fixkostenquote
* &#x20;Variable Kostenquote
* &#x20;Monatsvergleich
* &#x20;Anzeige der größten variablen Ausgabenkategorie
* &#x20;Automatische Finanzbewertung
* &#x20;Generierung von Spartipps



**Datenverwaltung**

* &#x20;Speicherung der letzten Berechnung
* &#x20;Speicherung von Monatswerten
* &#x20;Zurücksetzen der Monatsdaten
